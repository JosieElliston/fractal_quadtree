# README

work in progress.

drawing the metabrot set, caching samples in a quadtree. [some](https://github.com/JosieElliston/fractal_egui) [previous](https://github.com/JosieElliston/fractal_explorer) iterations.

## controls

- pan/zoom with mouse
- space to toggle getting new samples
- number keys to switch which fractal is drawn
- hold backtick to pan/zoom the secondary camera, which is used for some fractals
- left/right arrows to draw the debug gradient decent points
